### editing

- bold : Cmd+B
- italic : Cmd+I
- underline : Cmd+U
- strikethrough : Cmd+Shift+S
- subscript : Cmd+Shift+-
- superscript : Cmd+Shift++
- header 1 : Cmd+1
- header 2 : Cmd+2
- header 3 : Cmd+3

- unordered list : Cmd+L
- ordered list : Cmd+Shift+L
- blockquote : Cmd+Shift+.
- code-block : Cmd+Shift+C

- indent : Cmd+]
- outdent : Cmd+[

- clear formatting : Cmd+0

- hide sidebar : Cmd+\
