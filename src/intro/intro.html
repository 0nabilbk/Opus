Welcome to Opus

a minimal note-taking app.

1
Open a folder
The folder contents are shown in the sidebar. You can create, rename, and delete files there.
2
Create a note
Use the sidebar to create a new .note file. Click on it to open it in the editor.
3
Write
Start your work! You can find formatting options and their keyboard shortcuts in the menu.

That's it.

enjoy using Opus.
